Giveaway Shuffle

A Discord giveaway bot built for manual entries, scheduled closings, delayed drawings, live updates, and fair shuffled ticket-pool winner selection.

View Commands Privacy Policy Terms of Service

Initial Release

Giveaway Shuffle is currently in its initial release. Features may be improved, adjusted, or expanded over time as the bot continues to grow.

What Giveaway Shuffle Does

Manual Entries

Add contestants manually and assign each person a custom number of entries.

Scheduled Closing

Set when entries close so no new entries are accepted after the deadline.

Delayed Drawings

Draw immediately when entries close, or schedule the drawing for a later time.

Live Updates

The giveaway post updates with current contestants and entry totals.

Fair Drawing

Each entry acts as its own ticket in a shuffled ticket pool.

Multi-Server Ready

Giveaways are separated by server so communities do not interfere with each other.

How Entries Work

Each entry is treated like a separate ticket. For example:

  • Blue = 5 entries
  • Red = 3 entries
  • Green = 2 entries

This creates 10 total tickets in the drawing pool. The more entries someone has, the better their chance, while every ticket still has a chance to win.

Closing & Drawing Times

Giveaway Shuffle separates the entry closing time from the actual drawing time.

  • Entries can close and draw immediately.
  • Entries can close first, then draw later.
  • This works well for live streams, scheduled events, and delayed reveal drawings.

Commands

/screate Create a new giveaway using a popup form.
/sentry Add entries to an active giveaway.
/sdesc Add or update the giveaway description.
/sclose Close a giveaway manually.
/sdraw Draw winner(s) manually.
/slist View current entries privately.
/srefresh Refresh the public giveaway post.
/sdelete Delete a finished giveaway.
/shelp Show the command help list.

Privacy Policy

Giveaway Shuffle only stores the information needed to operate giveaways. This may include Discord server IDs, channel IDs, giveaway names, contestant names, entry counts, winner information, and timing details.

Giveaway Shuffle does not collect passwords, payment information, private messages, or unnecessary personal data.

Stored giveaway data is used only to run contests, update giveaway posts, and select winners.

Server administrators may request removal of stored giveaway data by contacting the bot operator.
