Welcome

Giveaway Shuffle was created for communities that need more control than basic reaction-entry bots. Hosts can control entry closing times, drawing times, contestant management, bulk imports, and live contest updates.

Features

Manual Entries

Add contestants and custom ticket counts instantly.

Bulk Importing

Add multiple contestants at once using /sbulk.

Scheduled Closing

Automatically stop entries at your chosen time.

Delayed Drawings

Close now, draw winners later for live reveals.

Live Updates

Giveaway posts refresh with totals and entries.

Time Editing

Adjust giveaway closing and draw times anytime.

Contestant Removal

Remove entrants and entries with /sremove.

Multi-Server Ready

Safe server-separated giveaways and data.

Fair Drawing System

Giveaway Shuffle uses a two-stage system that rewards more entries while keeping the final reveal exciting.

  • Every entry becomes a ticket.
  • More tickets improve Top 10 finalist odds.
  • Finalists then compete evenly in the last round.
  • Winner reveal happens live with suspense eliminations.

Commands

/screate
Create a new giveaway.
/sdesc
Edit giveaway description.
/sentry
Add entries manually.
/sbulk
Bulk add multiple contestants + entries.
/sremove
Remove a contestant and all entries.
/stime
Adjust close and drawing times.
/sclose
Close entries immediately.
/sdraw
Draw winner(s) now.
/slist
View current entries privately.
/srefresh
Refresh giveaway post.
/sdelete
Delete finished giveaway.
/shelp
Show help menu.
